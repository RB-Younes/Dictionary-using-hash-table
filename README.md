# Dictionary using Hash Table

This project implements a dictionary using a hash table data structure. It allows users to efficiently store and retrieve key-value pairs. The hash table provides fast access to entries, making it suitable for applications requiring quick lookups.

## Features

- Efficient storage and retrieval of key-value pairs
- Implemented using a hash table for fast access
- Supports adding, removing, and searching for entries
